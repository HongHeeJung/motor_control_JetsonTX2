# Reconstruct 3d map using 2d Lidar
## Development Enviornment
Dev. Environment:  Ubuntu 18.04, ROS melodic  
Language:  C++, Shell script  
Sensor:  RPLidar A3, Real-sense Camera T265  
Actuator:  Step motor (sbd-shp-42d0014)
